# ระบบแข่งขัน A-Math — Math Week 2026

ระบบกรอกคะแนนและจัดอันดับการแข่งขัน A-Math รองรับการกรอกพร้อมกันหลายเครื่องแบบ real-time
สร้างด้วย **Next.js + Supabase** deploy บน **Vercel**

## หน้าเว็บ
- `/` — หน้าแรก รวมลิงก์
- `/admin` — แอดมิน: จัดการผู้เข้าแข่งขัน, จัดคู่, ตารางอันดับ, เกียรติบัตร
- `/scoring` — กรรมการกรอกคะแนน (เปิดพร้อมกันหลายเครื่องได้)
- `/display` — จอแสดงผลตารางอันดับสด (ฉายขึ้นจอใหญ่)

## กติกาการนับคะแนน (พอร์ตจากระบบ Google Apps Script เดิม)
- ชนะ = 2 แต้ม, เสมอ = 1, แพ้ = 0
- ผลต่างต่อเกมจำกัด ±250 (เกมสุดท้าย ±200)
- ได้ BYE = ชนะอัตโนมัติ ผลต่าง +100
- จัดอันดับ: แต้มสะสม → ผลต่างสะสม → คะแนนดิบรวม
- เกียรติบัตร: 60% แรก (ปัดขึ้น)
- เกม 1 สุ่มจับคู่ / เกม 2 เป็นต้นไป King of the Hill (อันดับ 1 พบ 2, 3 พบ 4, ...)

## วิธีติดตั้งและรัน (ทีละขั้น)

### 1) เตรียมฐานข้อมูล Supabase
1. เข้า https://supabase.com → เปิดโปรเจกต์
2. เมนูซ้าย **SQL Editor** → **New query**
3. เปิดไฟล์ `supabase_schema.sql` ในโฟลเดอร์นี้ → คัดลอกทั้งหมดไปวาง → กด **Run**
   (ถ้ารันซ้ำแล้วบรรทัด `alter publication ... add table` ขึ้น error ว่ามีอยู่แล้ว ไม่เป็นไร ข้ามได้)

### 2) ตั้งค่า environment
ไฟล์ `.env.local` มีค่าให้แล้ว:
```
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
```

### 3) ติดตั้งและรันในเครื่อง
เปิด Terminal ในโฟลเดอร์นี้แล้วพิมพ์:
```
npm install
npm run dev
```
เปิดเบราว์เซอร์ไปที่ http://localhost:3000

### 4) ลำดับการใช้งานวันแข่ง
1. ไปหน้า `/admin` → เลือกระดับ (ม.ต้น/ม.ปลาย) → เพิ่มหรือนำเข้ารายชื่อผู้เข้าแข่งขัน
2. กด **เกม 1** เพื่อสุ่มจับคู่ → ดูรายการคู่/โต๊ะ
3. กรรมการเปิด `/scoring` (กี่เครื่องก็ได้) → เลือกเกม → กรอกเลขโต๊ะ → กรอกคะแนน → บันทึก
4. กลับ `/admin` → กด **โหลด/รีเฟรช** ดูตารางอันดับ → กด **เกม 2** (King of the Hill)
5. เปิด `/display` ฉายขึ้นจอใหญ่ — อัปเดตอันดับสดอัตโนมัติ
6. กด **เพิ่มเกม** ได้ถ้าต้องการรอบมากกว่าค่าเริ่มต้น 3 เกม

> **กรอกพร้อมกันไม่ชนกัน:** แต่ละโต๊ะคือข้อมูล 1 ช่องเฉพาะตัว ถ้ามีคนกรอกโต๊ะเดียวกันซ้ำ ระบบจะเตือนให้ยืนยันก่อนบันทึกทับ

## Deploy ขึ้น Vercel
1. push โค้ดขึ้น GitHub repo `amath-2026`
2. เข้า https://vercel.com → **Add New Project** → เลือก repo
3. ใส่ Environment Variables 2 ตัว (เหมือนใน `.env.local`)
4. กด **Deploy**
